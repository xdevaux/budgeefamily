# Guide : Mes achats CB

## Présentation
"Mes achats CB" permet de gérer vos achats par carte bancaire avec reconnaissance automatique des reçus (OCR).

## Fonctionnalités
- Upload multiple de reçus (photos ou scans)
- Extraction automatique des données (date, montant, commerçant)
- Catégorisation intelligente
- Validation avant enregistrement
- Synchronisation automatique avec la balance

## Comment utiliser

### 1. Ajouter des achats
1. Aller dans **Dépenses > Mes achats CB**
2. Cliquer sur **"Ajouter des achats"**
3. Sélectionner jusqu'à 10 photos/scans de reçus CB
4. Cliquer sur **"Traiter avec OCR"**

### 2. Validation
- Le système affiche une grille avec toutes les données extraites
- Vérifiez et corrigez si nécessaire
- Décochez les achats que vous ne voulez pas enregistrer
- Cliquez sur **"Enregistrer les achats"**

### 3. Consultation
- Tous vos achats apparaissent dans la liste
- Utilisez les filtres (catégorie, mois, année)
- Cliquez sur l'icône "œil" pour voir l'image du reçu

## Conseils pour de meilleurs résultats OCR
- Prenez des photos bien éclairées et nettes
- Évitez les ombres et reflets
- Cadrez le reçu correctement
- Préférez les scans aux photos si possible

## Catégories automatiques détectées
Le système reconnaît automatiquement les catégories suivantes :
- **Alimentation** : Carrefour, Auchan, Leclerc, Lidl, Intermarché, etc.
- **Carburant** : Total, BP, Shell, Esso, etc.
- **Restaurant** : McDonald's, KFC, restaurants, cafés, etc.
- **Transport** : SNCF, RATP, Uber, taxis, etc.
- **Santé** : Pharmacies, hôpitaux, cliniques, etc.
- **Loisirs** : Cinéma, théâtre, sport, etc.
- **Vêtements** : Zara, H&M, Kiabi, Decathlon, etc.
- **Bricolage** : Leroy Merlin, Castorama, etc.

## Limites
- Maximum 10 fichiers par upload
- Taille maximale : 5 Mo par fichier
- Formats acceptés : JPG, PNG, PDF
- La précision OCR dépend de la qualité de l'image

## Dépannage

### L'OCR ne détecte pas le montant
- Vérifiez que le montant total est clairement visible
- Essayez de recadrer l'image sur la partie avec le montant
- Vous pouvez toujours corriger manuellement dans la grille de validation

### Le nom du commerçant est incorrect
- Modifiez-le directement dans la grille de validation avant d'enregistrer
- Ou éditez l'achat après enregistrement

### La catégorie ne correspond pas
- Sélectionnez la bonne catégorie dans la liste déroulante de la grille
- Vous pouvez aussi modifier la catégorie après enregistrement
